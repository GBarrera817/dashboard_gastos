Dashboard de Gastos
Bienvenido. Antes de comenzar necesitas configurar tu propia base de datos en Google Sheets. Solo toma unos minutos.
1 Crea tu Google Sheet
  1. Ve a sheets.new y crea una hoja nueva
  2. Nómbrala como quieras, ej: Mis Gastos
  3. Copia el ID de la hoja desde la URL:
    https://docs.google.com/spreadsheets/d/[ESTE ES EL ID]/edit
✓ Las hojas se crearán automáticamente al ejecutar el setup del Apps Script
2 Configura Google Apps Script
  1. En tu Google Sheet ve a Extensiones → Apps Script
  2. Borra el código existente y pega el código del archivo apps_script.gs del repositorio
  3. Reemplaza SHEET_ID con el ID copiado en el paso anterior
  4. Reemplaza EMAIL_DESTINO con tu correo Gmail
  5. Guarda el proyecto (Ctrl+S)
  6. Ejecuta la función setup — esto crea todas las hojas automáticamente:
    • Selecciona setup en el selector de funciones
    • Clic en ▶️ Ejecutar
    • Autoriza los permisos que solicite
  7. Ve a Implementar → Nueva implementación
    • Tipo: Aplicación web
    • Ejecutar como: Yo
    • Acceso: Cualquier persona
  8. Copia la URL que termina en /exec
